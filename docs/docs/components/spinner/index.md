# Spinner

## Sizes

```hbs preview-template
<DemoSpaceX>
  <VelvetSpinner @size='xs' />
  <VelvetSpinner @size='sm' />
  <VelvetSpinner @size='md' />
  <VelvetSpinner @size='lg' />
  <VelvetSpinner @size='xl' />
</DemoSpaceX>
```

## Arguments

| Name    | Description              | Type     | Default Value |
| ------- | ------------------------ | -------- | ------------- |
| `@size` | The size of the spinner. | `String` | `'md'`        |
